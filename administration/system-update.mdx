# System Updates

Shiipp Courier frequently releases updates that include new features, bug fixes, and security patches. You can install these directly from your backoffice.

## Checking for Updates
1. Go to **Administration > System Update** (or look for the rocket icon with a red badge in the topbar).
2. The system will automatically check if a new version is available.
3. Review the changelog.
4. Click **Apply Update**.

The update process usually takes less than a minute. Your system will briefly enter maintenance mode during this time.
